| lg | 24px | Section padding (py-6, gap-6) |

| 2xl | 48px | Section padding (py-12, gap-12) |
| 3xl | 64px | Section padding (py-16, gap-16) |

**Phase 6 Timeline Spacing:**
- Timeline dot diameter: 12px (w-3 h-3)
- Dot-to-card gap: 16px (gap-4)
- Entry vertical spacing: 40px (space-y-10)
- Left padding for cards: 24px (pl-6) — provides room for timeline line + dot

## Copywriting Contract

| Element | Copy |
|---------|------|
| Section heading | `{t('sections.experience')}` — existing translation key "Experience" (EN) / "Experiencia" (ES) |
| Company name | From translation file: `experience.timeline[N].company` |
| Job title | From translation file: `experience.timeline[N].title` |

| Achievement bullets | From translation file: `experience.timeline[N].achievements[]` — 2-3 bullets per position |
| Empty state | Not applicable — all 12 positions will be displayed from LinkedIn data |
| Error state | Not applicable — static content, no API/dynamic failures |

**Copy constraints:**
- All content bilingual (EN/ES) via next-intl translation files
- Date format: "Month YYYY - Month YYYY" (e.g., "April 2025 - Present")
- Achievement bullets: natural sentence format, no character limits (LinkedIn content as-is)
- Spanish months from LinkedIn PDF preserved (abril, marzo, etc.) OR converted to English in translation files

## Component Structure

**Timeline Component:**
- Semantic HTML: `<ol>` ordered list (12 items in reverse chronological order)
- Flexbox layout: vertical single-column all breakpoints
- Timeline visual elements: CSS-only (no SVG assets)
  - Connector line: `border-l-2` absolute positioned or pseudo-element `::before`
  - Dots: `<span>` elements with `rounded-full w-3 h-3` positioned at each entry
- Responsive: spacing/typography scale via Tailwind breakpoints (sm:, lg:), NO layout changes

**TimelineEntry Component (per position):**
- Structure: Flexbox row with dot + content card
- Dot: Circle positioned on timeline line (absolute or flexbox)
- Card: Flex-1 content area with company, title, dates, bullets
- Achievement bullets: `<ul>` unordered list with `list-disc` markers
- No border, no shadow, no background (relies on Section alternating bg)

**Integration:**
- Replaces placeholder at `src/app/[lang]/page.tsx` line 53-60 (Experience section)
- Uses existing `<Section id="experience" alternate>` wrapper
- Translation data: `messages/en.json` and `messages/es.json` extended with `experience.timeline[]` array
