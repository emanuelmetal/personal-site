**1. Create `src/components/Header.tsx` (Client Component):**

This is the main navigation component. It MUST start with `'use client';` because it uses useState for mobile drawer toggle and useEffect + IntersectionObserver for active section detection.

```typescript
'use client';

import { useState, useEffect } from 'react';
import { useTranslations } from 'next-intl';
import MobileDrawer from './MobileDrawer';
```

**Constants:**
```typescript
const NAV_ITEMS = ['about', 'experience', 'skills', 'portfolio', 'contact'] as const;
```

**State:**
- `const [drawerOpen, setDrawerOpen] = useState(false);`
- `const [activeSection, setActiveSection] = useState<string>('about');`

**Active section detection with IntersectionObserver:**
```typescript
useEffect(() => {
  const observer = new IntersectionObserver(
    (entries) => {
      entries.forEach((entry) => {
        if (entry.isIntersecting) {
          setActiveSection(entry.target.id);
        }
      });
    },
    { rootMargin: '-20% 0px -80% 0px' }
  );
  NAV_ITEMS.forEach((id) => {
    const el = document.getElementById(id);
    if (el) observer.observe(el);
  });
  return () => observer.disconnect();
}, []);
```

The rootMargin `-20% 0px -80% 0px` triggers when a section is in the top 20% of the viewport, which gives natural active state transitions during scroll.

**Render structure:**
