<!-- Accessibility patterns to implement -->

WCAG AA Contrast Requirements:
- Normal text (< 24px or < 19px bold): 4.5:1 minimum
- Large text (≥ 24px or ≥ 19px bold): 3:1 minimum
- UI components (buttons, borders): 3:1 minimum

Common dark mode violations (need lighter text):
- `text-slate-400` on `bg-slate-950`: ~2.8:1 (fails) → use `text-slate-300`: ~5.1:1 (passes)
- `text-slate-500` on `bg-slate-950`: ~1.9:1 (fails) → use `text-slate-400`: ~3.8:1 (passes for large text)

Focus indicator pattern (per D-12):
```tsx
className="... focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-blue-500 focus-visible:ring-offset-2"
```

ARIA live region pattern (per D-16):
```tsx
<div
  role="status"
  aria-live="polite"
  aria-atomic="true"
  className="sr-only"
>
  {activeSection && `Now viewing: ${t(activeSection)}`}
</div>
```

Skip link pattern (per D-13):
```tsx
<a
  href="#main"
  className="skip-link"
>
  Skip to main content
</a>

/* CSS */
.skip-link {
  position: absolute;
  left: -9999px;
  z-index: 999;
}
.skip-link:focus {
  left: 50%;
  top: 1rem;
  transform: translateX(-50%);
  /* visible styles */
}
```
